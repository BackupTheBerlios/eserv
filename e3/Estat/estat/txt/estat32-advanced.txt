
  РАСШИФРОВКА ТАБЛИЦ СТАТИСТИКИ НА ПРИМЕРЕ HTTP ПРОТОКОЛА


  General Statistic (Главная или Общая статистика)
  ---------------------------------------------------------------------------
  all:                  Для детальной статистики - что именно ищем в логах 
                        (IP, имя пользователя, емайл, узел веб) и какой 
                        именно
  ---------------------------------------------------------------------------
  Server (protocol):    Обсчитываемый протокол
  ---------------------------------------------------------------------------
  Day(s):               Период, за который формируется статистика 
                        (сегодня, вчера, несколько дней)
  ---------------------------------------------------------------------------
  Period of Time:       Учитываемое в статистике время 
                        (напр. с 8:30 до 17:30)
  ---------------------------------------------------------------------------
  the whole Ip numbers: Всего уникальных IP адресов
  ---------------------------------------------------------------------------
  the whole Users:      Всего уникальных пользователей
  ---------------------------------------------------------------------------
  the whole Requests:   Всего запрошено уникальных веб-узлов
  ---------------------------------------------------------------------------
  the whole Login:      Всего обращений к серверу (запросов)
  ---------------------------------------------------------------------------
  the whole Kb:         Общий трафик (всего в килобайтах)
  ---------------------------------------------------------------------------
  the whole Time:       Общее время, потраченное на обработку запросов 
                        (ЧЧ:ММ:СС)
  ---------------------------------------------------------------------------


  Days Statistic (Статистика по дням)
  ---------------------------------------------------------------------------
  N?                    Порядковый номер
  ---------------------------------------------------------------------------
  day                   День
  ---------------------------------------------------------------------------
  login                 Всего в этот день запросов
  ---------------------------------------------------------------------------
  traffic               Общий трафик (всего в килобайтах)
  ---------------------------------------------------------------------------
  time                  Общее время, потраченное на обработку запросов 
                        (ЧЧ:ММ:СС)
  ---------------------------------------------------------------------------
  graphic               Наглядное представление (за 100% принимается 
                        наибольший показатель по строке loggin, остальные
                        отсчитываются относительно этого дня). 
                        В версиях старше 2.53 тип графика настраивается 
                        в файле конфигурации (Estat32.ini)
  ---------------------------------------------------------------------------
  %                     Численное представление процентов относительно 
                        наибольшего показателя (также как в graphic)
  ---------------------------------------------------------------------------


  Hour-to-hour Statistic (Почасовая статистика)
  ---------------------------------------------------------------------------
  hour                  Час (напр. 00:00-01:00)
  ---------------------------------------------------------------------------
  login                 Количество обращений к серверу в этот час
  ---------------------------------------------------------------------------
  traffic               Часовой трафик в килобайтах
  ---------------------------------------------------------------------------
  time                  Потраченное на обработку запросов время (ММ:СС)
  ---------------------------------------------------------------------------
  graphic               График (так же как в Days Statistic)
  ---------------------------------------------------------------------------
  %                     Числовое предстваление процентов (также)
  ---------------------------------------------------------------------------


  Statistics on ip addresses (Статистика по IP адресам)
  ---------------------------------------------------------------------------
  N?                    Порядковый номер адреса
  ---------------------------------------------------------------------------
* ip address            IP адрес
  ---------------------------------------------------------------------------
  login                 Запросов с этого адреса
  ---------------------------------------------------------------------------
  traffic               Трафик этого адреса (принято им килобайт)
  ---------------------------------------------------------------------------
  time                  Время на обработку запросов от этого IP адреса
  ---------------------------------------------------------------------------
  graphic               График (так же как в Days Statistic)
  ---------------------------------------------------------------------------
  %                     Числовое предстваление процентов (также)
  ---------------------------------------------------------------------------


  Statistics on users (Статистика по пользователям)
  ---------------------------------------------------------------------------
  N?                    Порядковый номер пользователя
  ---------------------------------------------------------------------------
* username              Имя пользователя
  ---------------------------------------------------------------------------
  login                 Обращений к среверу под этим именем
  ---------------------------------------------------------------------------
  traffic               Трафик для этого пользователя (в килобайтах)
  ---------------------------------------------------------------------------
  time                  Время на обработку запросов от этого пользователя
  ---------------------------------------------------------------------------
  graphic               График (так же как в Days Statistic)
  ---------------------------------------------------------------------------
  %                     Числовое предстваление процентов (также)
  ---------------------------------------------------------------------------


  Statistics on requests (Статистика по запросам, т.е. веб узлам)
  ---------------------------------------------------------------------------
  N?                    Порядковый номер веб узла
  ---------------------------------------------------------------------------
* request               Что именно запрошенно (узел веб, при формировании
                        статистики обратите внимание на ключевое слово
                        HTTPStyle в Estat32.ini)
  ---------------------------------------------------------------------------
  attempts              Кол-во попыток (запросов) этого узла веб
  ---------------------------------------------------------------------------
  traffic               Кол-во переданной информации (трафик в килобайтах)
  ---------------------------------------------------------------------------
  time                  Время на обработку запросов и передачу файлов
  ---------------------------------------------------------------------------
  graphic               График (так же как в Days Statistic)
  ---------------------------------------------------------------------------
  %                     Числовое предстваление процентов (также)
  ---------------------------------------------------------------------------


  Top Statistic (Статистика наивысших показателей)
  ---------------------------------------------------------------------------
  Days Counter top:     День с наивысшим кол-вом запросов к серверу +
                        + это само кол-во запросов
  ---------------------------------------------------------------------------
  Days Kb top:          День с наибольшим трафиком + сам трафик к Кб
  ---------------------------------------------------------------------------
  Days Time top:        День, в который потраченно больше всего времени на 
                        обработку запросов + само потраченное время ЧЧ:ММ:СС
  ---------------------------------------------------------------------------

  Hour-to-hour Counter top: Наиболее посещаемый час + кол-во заходов в 
                            течение часа
  ---------------------------------------------------------------------------
  Hour-to-hour Kb top:      Час с наибольшим трафиком + сам трафик в Кб
  ---------------------------------------------------------------------------
  Hour-to-hour Time top:    Час, в который больше всего времени потрачено на 
                            обработку запросов + это время ММ:СС
  ---------------------------------------------------------------------------

* ip numbers Counter top:   IP адрес, с которого было больше всего запросов
                            + сколько именно запросов
  ---------------------------------------------------------------------------
* ip numbers Kb top:        IP адрес, с наибольшим трафиком + показатель
                            трафика в килобайтах
  ---------------------------------------------------------------------------
* ip numbers Time top:      IP адрес, на обработку запросов с которого 
                            потраченно больше всего времени + это время
  ---------------------------------------------------------------------------

* user Counter top:         Пользователь, запросивший сервер больше других 
                            + кол-во этих запросов
  ---------------------------------------------------------------------------
* user Kb top:              Пользователь, лидирующий по трафику + показатель
                            его трафика в килобайтах
  ---------------------------------------------------------------------------
* user Time top:            Пользователь, лидирующий по времени + это время 
                            в ЧЧ:ММ:СС
  ---------------------------------------------------------------------------

* Requests Counter top:     Узел веб, запрошенный больше других + кол-во
                            запросов   
  ---------------------------------------------------------------------------
* Requests Kb top:          Узел, составивший большую часть трафика среди
                            других уникальных узлов веб + его трафик в Кб
  ---------------------------------------------------------------------------
* Requests Time top:        Узел веб, на обработку запросов и передачу
                            которого потраченно больше всего времени 
                            + это время ЧЧ:ММ:СС  
  ---------------------------------------------------------------------------

Ключевые слова, отмеченные звездочкой, являются ссылками, т.е. при нажатии
на них можно получить детальную статистику по каждому пользователю, ip адресу,
запрошенному узлу веб или емайлу.
