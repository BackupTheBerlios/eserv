Service plugin

В операционных системах Windows NT4, 2000, XP {PROGNAME} может работать системным сервисом - запускаться автоматически при включении компьютера, "невидимо", независимо от работы пользователей на этом компьютере.

Для установки {PROGNAME} в качестве сервиса нажмите Install service, для удаления - Uninstall service. Для запуска - Start service, для останова - Stop service. Учтите, что после останова сервиса прокси перестает работать - и вы не сможете его запустить обратно через этот веб-интерфейс, если ваш браузер настроен работать через останавливаемый прокси. Лучше настроить браузер так, чтобы с этим веб-интерфейсом он работал напрямую.

Также можно управлять сервисом из панели управления. Имя сервиса - {PROGFILE}.
