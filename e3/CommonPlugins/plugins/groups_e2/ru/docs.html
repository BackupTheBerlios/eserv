Группы пользователей Eserv/2

Если по каким-то причинам требуется использование групп пользователей из Eserv/2, не приводя их к новому формату, то можно использовать этот модуль. Он добавляет в список доступных правил правило "IsMember: имя_группы", которое возвращает истину, если текущий пользователь входит в указанную группу. Имя файла с группами задается командой "UsersGroupsFileE2: имя_файла".
