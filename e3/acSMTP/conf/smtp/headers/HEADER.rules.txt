\ Этот файл вычисляется для каждого заголовка письма.
\ Имя заголовка в HEADER-NAME, а полная строка - в HEADER-STR.
\ Можно менять заголовок командой SetHeader, например:
\ S" Subject: other text" SetHeader
\ Или удалять заголовок: S" " SetHeader

\ Удаление строк X-... из заголовков исходящей почты (просто пример)
\ HEADER-NAME 2 MIN S" X-" COMPARE 0= IsOutboundMail AND | S" " SetHeader \EOF

\ Выполняем файл обработки поля заголовка, если этот файл есть:
EvalRulesIfExists: smtp\headers\{HEADER-NAME}
