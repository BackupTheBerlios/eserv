Расширение acTCP функциями получения файлов по http.

Использование в активных страницах acWEB: 
{include: /local_uri} или {include: http://some_url}.
В случае обращения к локальным ресурсам используется то же имя и пароль,
что было использовано для текущей страницы. При включении страницы по
полному URL (с http) можно явно указать имя:пароль - 
http://user:pass@host:port/file.
