В эти комплекты программ входят отдельные программы и
компоненты, распространяемые с различными лицензиями:

 1. Почтовый сервер SMTP/POP3/IMAP4 (Eserv):
    включает программы acSMTP, acIMAP - OpenSource
 2. Прокси сервер HTTP/FTP/SOCKS/MAP (Eproxy):
    включает программы acFreeProxy, Eproxy - OpenSource
 3. Веб-сервер acWEB - OpenSource
 4. Файловый сервер acFTP - OpenSource
 5. Скриптер ForthScript, компилятор SPF - OpenSource
 6. Расширения, базовая конфигурация и средства
    интеграции компонентов - OpenSource/SH
 7. Антивирусные API - ShareWare
 8. Статистические программы - ShareWare
 9. Программы управления трафиком - ShareWare
10. Планировщик - ShareWare
11. FireWall API - OpenSource/SH
