Конфигурация безопасности proxy

Сейчас будет произведена попытка автоматической конфигурации безопасности proxy. При этом будет создан список локальных подсетей, доступ к proxy из которых будет разрешен. Подключения к прокси из внешних подсетей будут запрещены.

Для надежного определения, какой из IP-адресов вашего сервера является доступным извне, будет произведена одна попытка прямой связи с внешним веб-сервером. После этого будет создан файл с настройками безопасности ({S" http-proxy\ip-list" RulesFile STR@}). Права доступа к proxy вы можете настроить вручную более детально через интерфейс управления proxy. Данная автоматическая настройка предназначена для исключения неавторизованного доступа при первом обращении к ненастроенному серверу.

Результаты автоматической настройки:
