Если это поле заполнено, то кроме URL/site будет проверяться и имя пользователя на совпадение с заданной в этом поле маской. Правило срабатывает, если истинны оба условия. Таким образом это дает возможность задавать правила, действительные только для части пользователей.

Если способ авторизации - IP address, то с маской сравниваться будет не имя пользователя, а его IP-адрес. Например, можно в этом поле можно задать маску "10.1.1.*".

Если поле не заполнено, то правило действует для всех пользователей.
