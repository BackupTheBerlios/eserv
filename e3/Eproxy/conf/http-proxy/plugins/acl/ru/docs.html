Управление доступом

Это расширение заменяет и расширяет функциональность старых black_list и adv_block.

При подключении этого расширения доступ к URL и хостам регулируется 4мя списками:

  • HostBlackList - черный список хостов
  • HostWhiteList - исключения из черного списка хостов
  • UrlBlackList - черный список URL
  • UrlWhiteList - исключения из черного списка URL
Каждый элемент списка включает 4 поля:
  • URL или HOST (допустимы групповые символы '*' и '?')
  • Имя пользователя или группа
  • Время действия
  • Правила
