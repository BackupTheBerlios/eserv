TrafС ver 2.1   14.Jul.2002

Плагин управления трафиком для Eserv3 (acFreeProxy 
и любых серверов на основе ядра acTCP) 

Это расширение служит для управления/ограничения трафика пользователей. 
Оно включает две части
  - Bandwidth control, -для управления средней скоростью и приоритетами.
  - Quotas control, -квоты на объем передаваемой информации.

Описание см. docs\man.html

Детальное описание конфигурации приведено в файле  docs\man.config.txt
