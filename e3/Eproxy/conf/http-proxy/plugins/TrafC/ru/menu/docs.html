Plugin Traffic Control (TrafC)

ver 2.1, 14.Jul.2002

Сборка { S" ..\aid.txt" FILE}

Это расширение служит для управления/ограничения трафика пользователей.

TrafC включает две части:
  • Bandwidth control - для ограничения средней скорости и приоритезации.
  • Quotas control - квоты на объем передаваемой информации.

Здесь предоставлен интерфейс для описания виртуальных каналов и правил их назначения:

  • Bandwidth -каналы, обеспечиваюие ограничение средней скорости;
  • Quota -каналы, обеспечивающие квоты;
  • Rules - правил для распределения трафика по виртуальным каналам.

Все настройки сохраняются в текстовых файлах в каталоге TrafC\.
