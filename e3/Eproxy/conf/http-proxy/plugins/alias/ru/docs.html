Псевдонимы для быстрого доступа к часто используемым сайтам

Если этот модуль включен, то можно вести список сокращенных имен сайтов для быстрого набора адресов в браузере. Список хранится в файле conf\http-proxy\plugins\alias\OnRequest.rules.txt

Например: 

UrlAlias .r http://www.rambler.ru/
UrlAlias .y http://www.yandex.ru/
UrlAlias .d ftp://ftp.eserv.ru/pub/
Для доступа к Yandex.ru будет достаточно набрать ".y".
