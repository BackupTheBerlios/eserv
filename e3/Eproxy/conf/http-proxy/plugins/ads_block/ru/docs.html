Plugin блокировки рекламных баннеров

Это расширение служит для уменьшения внешнего трафика прокси и для ускорения загрузки страниц за счет исключения скачивания рекламы. Вместо рекламных картинок выдаются пустые картинки-заменители.

Автоматически отличить рекламу от нужных изображений невозможно, поэтому настраивается список рекламных URL и серверов.
