Источники авторизации

{PROGNAME} может брать списки пользователей и групп из различных источников:
  • Windows NT/2000/XP/.NET на локальной машине
  • Другого Windows-сервера (контроллера домена, Active Directory и т.п.)
  • Из файла
  • Из базы данных
  • Веб-сервера
  • И др. (способы авторизации устанавливаются plugin'ами)
Список источников авторизации и их параметров (имя сервера, имя файла, база данных и SQL-запрос, URL сервера и т.п.) настраивается в {PROGNAME} в разделе Источники авторизации. Можно использовать одновременно несколько источников.

Домены

С каждым источником можно сопоставить один или более доменов. Некоторые способы авторизации сами подразумевают наличие доменов, но не все. Поэтому явное указание доменов делает авторизацию единообразной, независимо от способа. В правилах авторизации можно указать, какой домен в каких случаях используется. По домену определяется соответствующий источник авторизации и собственно проводится авторизация в этом источнике. Соответственно в разных доменах могут быть различные списки пользователей (и пользователи с одинаковыми именами будут считаться различными и могут иметь разные пароли, права), а также могут быть общие для нескольких доменов списки - из одного источника (но права могут быть разными при необходимости).
