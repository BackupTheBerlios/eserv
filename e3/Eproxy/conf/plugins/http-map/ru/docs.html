HTTP-MAP

Этот модуль добавляет в программу возможность отображения любого TCP-порта (и любого TCP-протокола, кроме FTP) через другой http-proxy.

Используется в виде "HTTP-MAP: proxy_host:proxy_port target_host:target_port" в conf\OnConnect.rules.txt.

Пример: 

Port= 2525 | HTTP-MAP: proxy.isp.ru:3128 mail.ru:25
( Если клиент соединился с портом 2525, то перенаправить его соединение на сервер mail.ru:25 через прокси proxy.isp.ru:3128 )

Этот модуль может быть единственным средством использования любых протоколов, если на компьютере с {PROGNAME} нет прямого соединения с интернетом, а есть доступ только через http-proxy.
